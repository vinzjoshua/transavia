READ ME

Het concept is een simpele website waar de verhalen te vinden zijn. Deze verhalen worden op de overzichtspagina weergegeven. Op de overzichtspagina kan de gebruiker ook nog direct zoeken naar een verhaal. Als de gebruiker wilt browsen opde verhalenwebsite kan de gebruiker de filters gebruiken. De gebruiker kan een eigen account aanmaken op verhalen op te slaan. Deze verhalen kunnen ook gedownload worden om offline te kunnen lezen.

De html is opgebouwd in met een body, header, main en een uitgecommende footer. Deze verschillende tags hebben daarin weer andere html tags zodat ik de verhalen kan weergeven.

Ik heb soms classes gebruikt omdat ik er anders niet uit kwam. Ik heb zovaak mogelijk geprobeerd om classes te  vermijden.

Met javascript probeer ik de micro interacties op te roepen wanneer nodig. Deze bestanden zijn in de map javascript te vinden.
Ik heb verschillende bestanden met javascript aangemaakt zodat ik overzichtelijk te werk kon gaan.

Ik heb een grote css met css magie waardoor ik mijn html kon vormgeven.
